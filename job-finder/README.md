# JobFinder

JobFinder is a production-oriented job search web application built with a split architecture:

- Frontend: HTML, CSS, Vanilla JavaScript
- Backend: Django + Django REST Framework

## Repository Structure

- `frontend/` user interface, reusable components, and modular JavaScript
- `backend/` Django project configuration and domain apps
- `docs/` architecture, API contracts, and team conventions

## Development Workflow

1. Create a feature branch from `main`
2. Implement one focused feature with tests/docs updates
3. Open a pull request with reviewer checklist
4. Merge only after review and passing checks
